GarudaEats merupakan sistem manajemen pemesanan kantin kampus yang dikembangkan dengan bahasa pemrograman Java dan menerapkan prinsip Object-Oriented Programming (OOP). 
Platform ini mengintegrasikan berbagai fungsi utama, mulai dari manajemen daftar menu dan stok barang, 
hingga pemrosesan transaksi yang dilengkapi dengan fitur loyalitas berupa poin bagi pengguna di setiap pembelian yang sukses
